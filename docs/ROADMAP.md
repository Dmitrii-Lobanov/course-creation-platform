# Roadmap

## Phase 0 — Repository Foundation

Goal: make the repository professional, readable, and architecture-ready.

Deliverables:

- README
- MVP plan
- Architecture document
- Roadmap
- Engineering decisions log
- Testing strategy
- Issue and PR templates
- Environment example

## Phase 1 — Project Foundation

Goal: initialize the application foundation.

Deliverables:

- Next.js App Router setup
- TypeScript
- Tailwind CSS
- ESLint
- Basic app layout
- Feature-based folder structure
- Shared UI primitives folder
- Environment configuration

## Phase 2 — Course Draft Creation

Goal: create the first vertical slice.

Deliverables:

- Database connection
- Initial schema for users and courses
- Course creation validation
- Course creation action/API
- Instructor dashboard course list
- Course status badge
- Course builder placeholder route

## Phase 3 — Course Builder MVP

Goal: allow instructors to structure course content.

Deliverables:

- Module creation
- Lesson creation
- Lesson editor
- Course structure sidebar
- Reordering support
- Draft save behavior
- Empty/loading/error states

## Phase 4 — Publishing Workflow

Goal: implement real course publishing rules.

Deliverables:

- Publishing checklist
- Domain validation
- Publish action
- Unpublish action
- Course catalog visibility

## Phase 5 — Student Learning Flow

Goal: allow students to enroll and learn.

Deliverables:

- Published course catalog
- Course landing page
- Enrollment action
- Course player
- Lesson completion
- Progress calculation
- Resume learning

## Phase 6 — Instructor Analytics

Goal: give instructors basic insight into student progress.

Deliverables:

- Total enrollments
- Active students
- Average progress
- Completion count
- Recent enrollments

## Phase 7 — Portfolio Polish

Goal: make the app impressive and demo-ready.

Deliverables:

- Responsive UI polish
- Seed data
- Tests
- Deployment
- Architecture README improvements
- Demo video
- Case study draft

## Future Production Features

- AI-assisted course outline generation
- File and video uploads
- Payments
- Certificates
- Reviews and ratings
- Comments and Q&A
- Notifications
- Admin moderation
- Audit logs
- Advanced analytics events
- Team accounts
